Screen recording in the browser using only Javascript.

This is a simple sketching app that records the screen, captures the microphone and spits out a video. All in Javascript.

If you're unfamiliar with the state of Javascript, this is a cutting-edge technique. Normally to record the screen you would need to download specialist software - but not with the technique I have used here!

I was initially concerned that screen recording would be impossible, but a quick search led me to a couple of great projects:

  • RecordRTC - for recording videos in the browser
  • videoconverter.js - a javascript port of ffmpeg, for merging and converting seperate video and audio streams

Standing on the shoulders of these giants, I created khan-draw!
