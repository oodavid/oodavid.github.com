Some notes about Jekyll and my blog tempates:

  • The Page title is automatically added above the HTML content
  • As you can see in the source, the "YAML front-matter" contains a number of variables...
  • Standard variables:
    • layout: blog - selects the _layout to use, while I tend to use blog, it would be trivial to make new templates for more involved pages
    • title: "Hello World" - used on the page itself and in the HTML > TITLE tag
    • tags ... - a YAML list of tags to associate with the article, I've not (yet) made much use for these, but I shall
    • published: false - add this to draft pages, or to remove a page from the site
  • Custom variables:
    • github: http://... - link to a github repo related to this post
    • gist: http://... - link to a gist related to this post
  • Previous and Next links are automatically added
  • FB-Comments are also added
