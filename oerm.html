During my career I have strived to contribute to open-source software and to share my programs and ideas for the benefit of the community at large. In doing so I have developed a very powerful "core" set of tools that is Project: OERM.

Project: OERM is a system that allows me to manage complex, relational databases (the best kind) in a very straightforward manner, all I need to do is define the database schema in code and a seriously powerful set of tools are exposed to me that create VERY powerful websites and mobile apps.

Query Generator

I can demonstrate the concept of this feature with a real-world example. I was asked a non-trivial question by a client:

"We need to see all Sites related to Contracts that we (an Account) have worked on within a Region, searching for the word 'Durham' and Clustered onto a google map..."

Anyone familiar with SQL queries will know that this naive question would test even the most hardened database engineer. With OERM it's a walk in the park.
